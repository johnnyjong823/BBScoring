# BBScoring — 資料結構設計

## 1. 資料模型總覽

```
Game (比賽)
 ├── GameInfo         比賽基本資訊
 ├── Team[]           隊伍（2支）
 │    ├── TeamInfo    隊伍資訊
 │    └── Player[]    球員名單
 ├── Lineup[]         打序（雙方）
 ├── Inning[]         局數記錄
 │    └── HalfInning  上/下半局
 │         └── AtBat[] 打席記錄
 │              └── Pitch[] 逐球記錄
 ├── RunnerState      壘上跑者即時狀態
 ├── ActionHistory[]  操作歷史（Undo/Redo）
 └── GameStats        比賽統計
```

#### 操作類型列表
| type | 說明 |
|------|------|
| `RECORD_PITCH` | 記錄投球結果 |
| `RECORD_HIT_RESULT` | 記錄打擊結果 |
| `RECORD_RUNNER_EVENT` | 記錄跑壘事件 |
| `CHANGE_PITCHER` | 更換投手 |
| `SUBSTITUTE_PLAYER` | 替換球員 |
| `MODIFY_PLAY` | 修改歷史記錄 |

## 6. 統計計算公式

### 6.1 打擊統計
```
打數 (AB) = 打席數 - BB - HBP - SAC - SF - IBB - INT
打擊率 (AVG) = H / AB
上壘率 (OBP) = (H + BB + HBP) / (AB + BB + HBP + SF)
長打率 (SLG) = 壘打數 / AB
壘打數 = 1B×1 + 2B×2 + 3B×3 + HR×4
OPS = OBP + SLG
```

### 6.2 投手統計
```
投球局數 (IP) = 出局數 / 3（顯示為 X.Y 格式，如 6.2 = 6又2/3局）
自責分率 (ERA) = (ER × 9) / IP
WHIP = (BB + H) / IP
K/9 = (K × 9) / IP
BB/9 = (BB × 9) / IP
```

### 6.3 守備統計
```
守備率 (FP) = (PO + A) / (PO + A + E)
```
